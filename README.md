# ZeroNews：零点新闻发布系统

## 1.ui设计

### （1）项目搭建

1. 新建项目

   > vue init webpack ZeroNews

2. 导入element-ui

   > npm i element-ui -S

   main.js中使用：

   ```js
   //导入element-ui
   import ElementUI from 'element-ui'
   import 'element-ui/lib/theme-chalk/index.css'
   
   Vue.use(ElementUI)
   ```

3. 导入axios

   > npm install axios --save

4. 修改端口号

   conf/index.js
